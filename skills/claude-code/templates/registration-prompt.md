# App Registration Template

Use this template when prompting the builder for registration details.

## Required Information

- **App Name**: _the name of the application_
- **Description**: _what the app does, in plain language_
- **Builder Name**: _your full name_
- **Builder Email**: _your organizational email_
- **Department**: _your department or team_
- **Build Tool**: _claude-code_ (auto-detected)
- **App Type**: _html-app | script | cli-tool | api-service | automation | other_
- **Tech Stack**: _list of technologies used_
- **Estimated Users**: _number of people who use this_
- **Status**: _active | experimental_
